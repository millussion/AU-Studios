#  AU STUDIOS - Photography Portfolio

Welcome to the **AU STUDIOS** repository! This is a professional, minimalist landing page designed for a photography studio based in Barranquilla, Colombia. The website serves as a visual gateway for clients seeking high-end digital storytelling and artistic photography.

---

##  About the Project

**AU STUDIOS** views photography as the art of freezing time and preserving the essence of a moment. The site is built with a focus on clean aesthetics, modern typography, and a semantic structure to ensure both a great user experience and SEO optimization.

###  Key Features
* **Semantic HTML5:** Full use of modern tags (`<header>`, `<main>`, `<section>`, `<aside>`, `<article>`) for better accessibility and structure.
* **Premium Typography:** Integrated Google Fonts featuring *Cascadia Mono*, *Miranda Sans*, and *Noto Sans*.
* **Service Showcase:** A dedicated grid section with custom SVG icons detailing business offerings.
* **Location & Contact:** Integrated Google Maps, operational hours, and direct social media links.
* **Cinematic Aesthetic:** Visual references to iconic cinematography (e.g., *The Lighthouse*, *Akira Kurosawa*, and *Raging Bull*).

---

##  Technologies Used

* **HTML5:** Core structure and layout.
* **CSS3:** Custom styling and responsive design (linked via `index.css`).
* **Google Fonts:** For high-quality web typography.
* **SVG:** Scalable vector graphics for high-definition icons and social links.
